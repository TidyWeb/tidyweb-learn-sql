Precision First

Good Habit

Q - What is the better default habit when I write a query?

Use SELECT with named columns by default. Keep SELECT * for prompts that explicitly ask for every column, or for quick table checks while practising.

Select Basics

Select Basics

Q - Show every column for every tomato.

SELECT * FROM tomatoes;
Select Basics

Q - Show only variety and type.

SELECT variety, type FROM tomatoes;
Select Basics

Q - Show only the distinct tomato types.

SELECT DISTINCT type FROM tomatoes;

Filtering

Filtering

Q - Find all roma tomatoes.

SELECT variety FROM tomatoes WHERE type = 'Roma';
Filtering

Q - Find tomatoes with a sweet taste profile.

SELECT variety, taste_profile FROM tomatoes WHERE taste_profile LIKE '%Sweet%';
Filtering

Q - Find tomatoes that are not beefsteak.

SELECT variety, type FROM tomatoes WHERE type != 'Beefsteak';
Filtering

Q - Find varieties starting with S.

SELECT variety FROM tomatoes WHERE variety LIKE 'S%';
Filtering

Q - Find tomatoes with sugar levels between 7.0 and 9.0 Brix.

SELECT variety, sugar_brix FROM tomatoes WHERE sugar_brix BETWEEN 7.0 AND 9.0;
Filtering

Q - Display variety and type for Cherry or Roma tomatoes. Use OR to combine the two conditions.

SELECT variety, type FROM tomatoes WHERE type = 'Cherry' OR type = 'Roma';

Sorting And Limit

Sorting And Limit

Q - Show the 4 sweetest tomatoes.

SELECT variety, sugar_brix FROM tomatoes ORDER BY sugar_brix DESC LIMIT 4;
Sorting And Limit

Q - Show the 3 largest tomatoes by diameter.

SELECT variety, diameter_cm FROM tomatoes ORDER BY diameter_cm DESC LIMIT 3;
Sorting And Limit

Q - Show the 5 cheapest tomatoes per kilo.

SELECT variety, cost_per_kg FROM tomatoes ORDER BY cost_per_kg ASC LIMIT 5;
Sorting And Limit

Q - Show the 4 most expensive tomatoes per kilo.

SELECT variety, cost_per_kg FROM tomatoes ORDER BY cost_per_kg DESC LIMIT 4;
Filter, Sort And Limit

Q - Display variety and sugar_brix for Cherry tomatoes, highest sugar level first, and return only the top 3.

SELECT variety, sugar_brix FROM tomatoes WHERE type = 'Cherry' ORDER BY sugar_brix DESC LIMIT 3;
