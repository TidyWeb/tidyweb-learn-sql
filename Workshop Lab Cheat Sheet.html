Fixed Practice Prompts

Insert

Q - Insert this complete row: item_id 9, Panasonic S5II, Camera, zone A4, Excellent, rate 68, rented 0 times, service due in 30 days, active Y.

INSERT INTO equipment
  (item_id, item_name, category, location_zone, condition_level,
   hourly_rate, times_rented, service_due_days, active_flag)
VALUES
  (9, 'Panasonic S5II', 'Camera', 'A4', 'Excellent',
   68, 0, 30, 'Y');
Update

Q - Increase hourly_rate by 10 for every row where category = 'Camera'.

UPDATE equipment
SET hourly_rate = hourly_rate + 10
WHERE category = 'Camera';
Update

Q - Set service_due_days to 0 for every row where it is currently 4 or fewer.

UPDATE equipment
SET service_due_days = 0
WHERE service_due_days <= 4;
Delete

Q - Delete only the row where item_id = 6 and item_name = 'Nanlite Tube Kit'.

DELETE FROM equipment
WHERE item_id = 6
  AND item_name = 'Nanlite Tube Kit';
Alter Table

Q - Add a text column named supplier_name to equipment.

ALTER TABLE equipment
ADD supplier_name TEXT;
Create Table

Q - Create maintenance_log with exactly: log_id INTEGER PRIMARY KEY, item_id INTEGER NOT NULL, and note TEXT NOT NULL.

CREATE TABLE maintenance_log (
  log_id INTEGER PRIMARY KEY,
  item_id INTEGER NOT NULL,
  note TEXT NOT NULL
);
Drop Table

Q - Remove the pre-seeded disposable table named obsolete_import. Do not change equipment.

DROP TABLE obsolete_import;

The destructive-looking tasks only affect the temporary in-browser practice database. Reset Database returns the original Workshop state.
