Tomato SQL Practice

This page gives you a tomato dataset with species, type, variety, taste profile, sugar level, size, and price. The tomato categories are based on common distinctions such as cherry, grape, roma/plum, beefsteak, and heirloom. The numeric values are simplified practice data so you can focus on SQL, not market accuracy.

What This Lab Is For

  • Practise basic SELECT queries on a food dataset instead of sport or film.
  • Filter both text categories and numeric growing or buying traits.
  • Get used to tables where some columns are descriptive and some are measurable.

How The Categories Work

The main type labels here follow common tomato groupings used in extension and produce guides: cherry, grape, roma/plum, beefsteak, and heirloom.

type variety taste_profile sugar_brix diameter_cm cost_per_kg
