Relationships SQL Practice

This is the point where SQL stops being one flat table. The key question is not just how to join: it is whether a film with no studio match should disappear or stay visible with a missing value.

What This Lab Is For

  • Match a film to its studio through the shared studio_id.
  • See the difference between keeping only matches and keeping every film.
  • Find the rows that do not have a match using IS NULL.

Dataset And Schema

Visible Data Table: films (bridge: studio_id) Keep this in view while you write queries.
film_idtitlegenrestudio_id
Visible Data Table: studios (bridge: studio_id) Match this column to films.studio_id.
