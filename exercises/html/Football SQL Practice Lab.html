Football SQL Practice

This page gives you a familiar football-player dataset and a live SQL box to practise the same early skills you have just covered in the tutorial. The key improvement over the original setup is that the table and the column names stay on the page, so you do not have to keep re-running SELECT * just to remember the structure.

What This Lab Is For

  • Practise basic SELECT queries with a table that feels familiar.
  • Filter by club, player name, goals, yellow cards, and market value.
  • Sort results and limit them without losing sight of the table structure.

Practice Prompts

        

      

      

        

          Select Basics
          
Try these first:

          
    
            
  • Show every column for every player.
    • 
            
  • Show only player_name and club.
    • 
            
  • Show only the distinct club names.
    • 
          

        

        

          Filtering
          
Then move on to filtering:

          
    
            
  • Display player_name for players from Arsenal.
    • 
            
  • Display the player_name and goals scored total for all players who have scored more than 18 goals.
    • 
            
  • Display player_name and club for players not at Liverpool.
    • 
            
  • Display player_name for players whose names start with Mo.
    • 
            
  • Display player_name and goals for Arsenal players with at least 15 goals. Use AND to combine the two conditions.
    • 
          

        

        

          Sorting And Limit
          
Then shape the answer:

          
    
            
  • Display player_name and goals for the top 5 scorers.
    • 
            
  • Display player_name and market_value_m for the 3 highest market values.
    • 
            
  • Display player_name and yellow_cards, with the lowest yellow card totals first.
