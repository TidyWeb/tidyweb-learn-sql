Precision First

Good Habit

Q - What is the better default habit when I write a query?

Use SELECT with named columns by default. Keep SELECT * for prompts that explicitly ask for every column, or for quick table checks while practising.

Select Basics

Select Basics

Q - Show every column for every film.

SELECT * FROM films;
Select Basics

Q - Show only title and release_year.

SELECT title, release_year FROM films;
Select Basics

Q - Show only the distinct genres.

SELECT DISTINCT genre FROM films;

Filtering

Filtering

Q - Find all science fiction films.

SELECT title FROM films WHERE genre = 'Science Fiction';
Filtering

Q - Find films directed by Christopher Nolan.

SELECT title FROM films WHERE director = 'Christopher Nolan';
Filtering

Q - Find films not directed by Denis Villeneuve.

SELECT title, director FROM films WHERE director != 'Denis Villeneuve';
Filtering

Q - Find film titles starting with The.

SELECT title FROM films WHERE title LIKE 'The%';
Filtering

Q - Find films released between 1990 and 2010.

SELECT title, release_year FROM films WHERE release_year BETWEEN 1990 AND 2010;
Filtering

Q - Display title and genre for Action or Science Fiction films. Use IN with those two genres.

SELECT title, genre FROM films WHERE genre IN ('Action', 'Science Fiction');

Sorting And Limit

Sorting And Limit

Q - Show the top 5 highest-rated films.

SELECT title, imdb_rating FROM films ORDER BY imdb_rating DESC LIMIT 5;
Sorting And Limit

Q - Show the 3 longest films.

SELECT title, runtime_minutes FROM films ORDER BY runtime_minutes DESC LIMIT 3;
Sorting And Limit

Q - Show the most recent 4 films.

SELECT title, release_year FROM films ORDER BY release_year DESC LIMIT 4;
Sorting And Limit

Q - Show the 5 biggest box offices.

SELECT title, box_office_m FROM films ORDER BY box_office_m DESC LIMIT 5;
Sorting, Limit And Offset

Q - Display title after sorting titles A to Z, skip the first 5 films, and return the next 5.

SELECT title FROM films ORDER BY title ASC LIMIT 5 OFFSET 5;
