Precision First

Good Habit

Q - What is the better default habit when I write a query?

Use SELECT with named columns by default. Keep SELECT * for prompts that explicitly ask for every column, or for quick table checks while practising.

Q - Find trips with a package cost above the overall average.

SELECT traveller_name, destination, package_cost FROM trips WHERE package_cost > (SELECT AVG(package_cost) FROM trips);
Subqueries

Q - Find trips with more nights than the average trip.

SELECT traveller_name, destination, nights FROM trips WHERE nights > (SELECT AVG(nights) FROM trips);

Q - Find trips whose review score is below the table average.

SELECT traveller_name, destination, review_score FROM trips WHERE review_score < (SELECT AVG(review_score) FROM trips);

Grouped Thinking

Grouped Thinking

Q - Display travel_style and average package cost by travel style. Name the average column average_package_cost.

SELECT travel_style, AVG(package_cost) AS average_package_cost FROM trips GROUP BY travel_style;
Grouped Thinking

Q - Display destination and average review score. Name the average column average_review_score, and show only destinations above 8.5.

SELECT destination, AVG(review_score) AS average_review_score FROM trips GROUP BY destination HAVING AVG(review_score) > 8.5;
Grouped Thinking

Q - Display booking_channel and total traveller count. Name the total column total_travellers, and show only booking channels above 8.

SELECT booking_channel, SUM(travellers) AS total_travellers FROM trips GROUP BY booking_channel HAVING SUM(travellers) > 8;

Logical Query Order

Where, Group By, Order By

Q - First keep only Winter trips. Then display booking_channel and the sum of travellers, named total_travellers. Group by channel and sort the totals highest first.

SELECT booking_channel, SUM(travellers) AS total_travellers FROM trips WHERE season = 'Winter' GROUP BY booking_channel ORDER BY total_travellers DESC;
Where, Group By, Having, Order By

Q - First keep only Website bookings. Then display travel_style and average package cost, named average_package_cost. Keep only averages above 500, then sort highest first.

SELECT travel_style, AVG(package_cost) AS average_package_cost FROM trips WHERE booking_channel = 'Website' GROUP BY travel_style HAVING AVG(package_cost) > 500 ORDER BY average_package_cost DESC;
Full Order With Limit

Q - Keep trips with review scores of at least 8. Display destination and total travellers, named total_travellers. Keep totals of at least 2, sort totals high to low then destination A–Z, and return the first 3 groups.

SELECT destination, SUM(travellers) AS total_travellers FROM trips WHERE review_score >= 8 GROUP BY destination HAVING SUM(travellers) >= 2 ORDER BY total_travellers DESC, destination ASC LIMIT 3;

Union

Q - Return all winter trips and all luxury trips using UNION.

SELECT traveller_name, destination, season FROM trips WHERE season = 'Winter' UNION SELECT traveller_name, destination, season FROM trips WHERE travel_style = 'Luxury';
Union

Q - Return all trips to Rome and all trips booked through agents.

SELECT traveller_name, destination, booking_channel FROM trips WHERE destination = 'Rome' UNION SELECT traveller_name, destination, booking_channel FROM trips WHERE booking_channel = 'Agent';
Union

Q - Try the same query with UNION ALL.

SELECT traveller_name, destination, season FROM trips WHERE season = 'Winter' UNION ALL SELECT traveller_name, destination, season FROM trips WHERE travel_style = 'Luxury';
