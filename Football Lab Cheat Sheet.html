Good Habit

Q - What is the better default habit when I write a query?

Use SELECT with named columns by default. Keep SELECT * for prompts that explicitly ask for every column, or for quick table checks while practising.

Select Basics

Select Basics

Q - Show every column for every player.

SELECT * FROM players;
Select Basics

Q - Show only player_name and club.

SELECT player_name, club FROM players;
Select Basics

Q - Show only the distinct club names.

SELECT DISTINCT club FROM players;

Filtering

Filtering

Q - Find players from Arsenal.

SELECT player_name FROM players WHERE club = 'Arsenal';
Filtering

Q - Find players with more than 18 goals.

SELECT player_name, goals FROM players WHERE goals > 18;
Filtering

Q - Find players not at Liverpool.

SELECT player_name, club FROM players WHERE club != 'Liverpool';
Filtering

Q - Find players whose names start with Mo.

SELECT player_name FROM players WHERE player_name LIKE 'Mo%';
Filtering

Q - Display player_name and goals for Arsenal players with at least 15 goals. Use AND to combine the two conditions.

SELECT player_name, goals FROM players WHERE club = 'Arsenal' AND goals >= 15;

Sorting And Limit

Q - Show the top 5 scorers.

SELECT player_name, goals FROM players ORDER BY goals DESC LIMIT 5;
Sorting And Limit

Q - Show the 3 highest market values.

SELECT player_name, market_value_m FROM players ORDER BY market_value_m DESC LIMIT 3;
Sorting And Limit

Q - Show the lowest yellow card totals first.

SELECT player_name, yellow_cards FROM players ORDER BY yellow_cards ASC;
