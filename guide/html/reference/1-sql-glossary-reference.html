Reference 1

SQL Glossary Reference

This is the quick-reference version of the core beginner SQL language. Use it when you remember the general idea but need to see the exact written form again.

SELECT

Choose what to show

Useful beginner translation: read SELECT as show me.

SELECT title, release_year
FROM

Name the table

Tells SQL where the columns are coming from.

FROM films

Filtering

WHERE

Keep only matching rows

WHERE release_year = 2014
=

Exact match

WHERE director = 'Christopher Nolan'
!=

Not equal

Use this for “not this exact value.”

WHERE club != 'Liverpool'
AND / OR

Combine conditions

WHERE genre = 'Science Fiction'
AND release_year > 2000
BETWEEN

Inclusive range

WHERE release_year BETWEEN 2000 AND 2010
IN / NOT IN

Match against a list

WHERE genre IN ('Drama', 'Crime')

Text Matching

'text'

Text needs quotes

'Christopher Nolan'
LIKE

Pattern match

WHERE title LIKE 'The%'
%

Any run of characters

'WALL-%'
_

Exactly one character

'WALL-_'

Sorting And Trimming

ORDER BY

Sort the result

ORDER BY release_year
ASC / DESC

Choose the direction

ORDER BY imdb_rating DESC
LIMIT

Keep only this many rows

LIMIT 5
OFFSET

Skip rows first

LIMIT 5 OFFSET 10

Joins And Missing Data

INNER JOIN

Keep matches only

FROM films
INNER JOIN boxoffice
ON films.film_id = boxoffice.film_id
LEFT JOIN

Keep the left table

FROM films
LEFT JOIN boxoffice
ON films.film_id = boxoffice.film_id
NULL

Missing value

WHERE distributor_id IS NULL
IS NOT NULL

Value is present

WHERE rating IS NOT NULL

Calculations And Grouping

AS

Name a result

release_year AS year_released
COUNT / AVG / SUM

Aggregate functions

SELECT COUNT(*), AVG(runtime_minutes)
FROM films
GROUP BY

Summarise by category

GROUP BY director
HAVING

Filter grouped results

HAVING COUNT(*) > 1

Writing Queries

INSERT INTO

Add rows

INSERT INTO films (title, director)
VALUES ('Example', 'Example')
UPDATE / SET

Change existing rows

UPDATE films
SET director = 'Correct Name'
WHERE film_id = 4
DELETE FROM

Remove rows

DELETE FROM films
WHERE film_id = 12
CREATE / ALTER / DROP

Change structure

CREATE TABLE awards (...)
ALTER TABLE awards ADD category TEXT
DROP TABLE awards

The Short Version

  • SELECT = show me
  • WHERE = keep only these rows
  • LIKE = pattern match, not exact match
  • ORDER BY = sort first
  • LIMIT = keep only this many rows
  • GROUP BY = change from row thinking to category thinking
